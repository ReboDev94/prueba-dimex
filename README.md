# Prueba Técnica

## Configuración

Instalar las dependencias
```bash
npm install
```

## Servidor de desarrollo

El servidor de desarrollo se inicia en  `http://localhost:3000`:

```bash
npm run dev
```

## Construir el build para producción

```bash
npm run build
```
